# Setup Checklist

## Supabase

1. Create a new Supabase project on the free plan.
2. Open the SQL editor.
3. Run `database/schema.sql`.
4. Run `database/seed.sql`.
5. Save the following for the app:
   - Project URL
   - Anon/public key

## Vercel

1. Create a new Vercel project.
2. Connect this Git repository when the Next.js app is scaffolded.
3. Use the free Hobby plan for v1.
4. Start with the default `vercel.app` URL.

## Optional Custom Domain

1. Buy a domain later if desired.
2. Connect it in Vercel once the app is deployed.

## Immediate Next Build Step

Scaffold the Next.js app in this repository and wire it to the Supabase schema.
